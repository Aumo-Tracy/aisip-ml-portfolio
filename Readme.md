# African Crisis Dataset EDA

This project performs exploratory data analysis on an African economic crisis dataset.

## Steps performed
- Loaded dataset with pandas
- Explored using .shape, .info(), .describe()
- Cleaned missing values and duplicates
- Created 5 visualizations:
  - Histogram
  - Scatter plot
  - Bar chart
  - Box plot
  - Correlation heatmap

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab
